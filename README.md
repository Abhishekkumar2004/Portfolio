## 🚀 Local Development Setup

To run this React & TypeScript portfolio locally in your VS Code environment:

### **1. Install Dependencies**
Open your terminal inside the project directory and run:
```bash
npm install
```

### **2. Run the Development Server**
Start the live reload server:
```bash
npm run dev
```
The server will boot and open the application locally on your designated port (e.g., `http://localhost:3000`).

### **3. Build for Production**
To generate optimized production bundle static files inside the `dist/` directory, execute:
```bash
npm run build
```

### **4. Run Code Linter**
Ensure your code complies with modern lint guidelines and contains zero TypeScript compile issues:
```bash
npm run lint
```
